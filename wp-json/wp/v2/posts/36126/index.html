BendTECH is hosting the second annual One Tech Meetup to Rule Them All, a meetup of all the meetup groups. If you’re interested in tech and\/or startup stuff, then this event is for you. If you’re new to town, it’s a great way to meet people with similar interests. And if you like beer and prizes, then it’s a great way to snag both of those things too.

The details:

One Tech Meetup<\/p>\n\n\n\n
